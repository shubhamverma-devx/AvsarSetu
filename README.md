AvsarSetu is an AI-powered platform designed to bridge the information and accessibility gap between students and government internship opportunities in India. Developed as a high-fidelity prototype for the Smart India Hackathon (SIH), it aims to democratize access to public sector roles for students across all tiers of the country.

ğŸŒŸ Key Features

1. AI-Powered Internship Co-Pilot
An intuitive, natural-language search experience that goes beyond simple keywords. It understands user intent to find the most relevant roles in various government departments.

2. Intelligent Resume Builder
A dedicated tool where students can enter their academic and professional details to generate professional resumes instantly.
Live Preview: Real-time updates as you type.
Multiple Templates: Switch between Classic, Modern, and Minimalist designs.
PDF Export: Ready-to-use resumes for official applications.

3. Dynamic Skill Report & Analytics
Uses data visualization (Chart.js) to provide students with a "Strength Analysis" across different domains like IT, Policy, and Management. It identifies skill gaps based on current government demand.

4. Smart Upskilling Recommendations
Based on the identified skill gaps, the platform suggests curated courses from verified platforms like NPTEL and SWAYAM to make the student "Internship Ready."

5. Multi-Language Support
To ensure inclusivity, the platform supports translation into 22+ Regional Indian Languages, including Hindi, Tamil, Telugu, Marathi, and Bengali.

6. Interactive Dashboard & Notifications
A personalized space for students to track saved internships, receive updates on application deadlines, and discover "For You" recommendations.

7. Dark Mode Support
A complete system-wide dark theme toggle accessible via settings for enhanced user experience.
